# cn-index-guide

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖各类信息索引、使用指南与参考资料。所有页面均为静态 HTML 文件，可由用户直接打开浏览，无需后端服务支持。

仓库不针对任何特定域名或网站，仅作为 HTML 文档的集合与发布平台。

## 目录说明

```
cn-index-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、图片等）
├── index.html      # 仓库首页，列出所有归档页面的链接
└── README.md       # 本文件
```

- **pages/** ：每个 HTML 文件对应一个独立页面，命名清晰，便于查找。
- **assets/** ：存放页面所需的样式文件、图标、字体等资源，保持页面结构整洁。
- **index.html** ：提供页面列表，方便快速浏览所有归档内容。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不依赖外部数据库或 API。
- 页面内容可能涉及工具使用、信息整理、操作指引等常见主题，不包含任何商业推广或诱导点击元素。
- 页面设计以简洁清晰为主，优先保证信息可读性。
- 归档内容可能随时间更新或调整，但不会针对特定网站或服务进行优化。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可浏览。
3. 如需部署到静态服务器，请确保所有文件路径保持相对关系。
